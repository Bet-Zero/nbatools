## Archive Scope

- Archive wave: Return Package Archive Sweep Wave 1.

- Destination directory:

- Root archive directory introduced: no.

## Candidate Rule

Wave 1 moved only package files matching this rule:

```text
all return_packages/raw-product/*.md
minus exact package paths linked from docs/, README.md, and AGENTS.md
```

The linked package manifest was generated from:

```text
rg --no-filename -o "return_packages/raw-product/[A-Z0-9_]+_RETURN_PACKAGE\.md" docs README.md AGENTS.md | sort -u
```

The archive candidate manifest was generated by subtracting that linked set
from the raw-product package inventory. The linked set and candidate set had no
overlap.
