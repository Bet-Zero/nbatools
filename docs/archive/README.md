# Docs Archive

This directory stores completed or superseded planning artifacts.

## Structure

- `completed-plans/`: closed plan-level docs.
- `completed-work-queues/`: historical work queues tied to closed plans/phases.
- `handoffs-and-boundaries/`: historical handoff notes and source-boundary decisions.
- `inventories/`: historical inventory/recon docs tied to closed phases.
- `product-polish/`: product-polish historical artifacts (completed queues and inventories).

## Rule

Archived docs are historical reference only.

Use `docs/planning/product_polish_master_plan.md` as the active continuation authority for product-polish work.
